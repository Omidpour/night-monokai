# Night Monokai Theme

A beautiful, Night Monokai-inspired theme for Visual Studio Code.
This theme is based on One Monokai by Joshua Azemoh.

## Installation

1. Open **Extensions** sidebar panel in Visual Studio Code. `View → Extensions`
2. Search for `Night Monokai`
3. Click **Install**
4. Click **Reload** to reload the your editor
5. Code > Preferences > Color Theme > **Night Monokai**
