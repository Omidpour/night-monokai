## ⚡ Installation

1. Open **Extensions** in VS Code
   → `View → Extensions`
2. Search for:

   ```
   Night Monokai Theme
   ```
3. Click **Install**
4. Reload VS Code (if needed)
5. Go to:

   ```
   Code → Preferences → Color Theme → Night Monokai
   ```
